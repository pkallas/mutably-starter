This app was built using Learner's Guild Mutably Heroku app. By filling out the first form, you can search for a Pokemon by it's id. By filling out the remaining forms, you can add a new Pokemon to the table. The "See All Pokemon" button can be used to see all Pokemon in the table, and the arrow to the right of the "See All Pokemon" button can be used to collapse the table.
